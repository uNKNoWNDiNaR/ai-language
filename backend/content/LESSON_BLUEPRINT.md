# Lesson Blueprint (A1 Content Rules)

This document is the source of truth for lesson content quality and structure.

## Core Rules
- One concept per question.
- A1 sentences: max 10–12 words.
- Avoid subordinate clauses early.
- Prefer present simple before past/future.
- Avoid rare vocab; reuse common words.

## Required Fields Per Question
- id
- conceptTag
- question
- answer
- acceptedAnswers
- hints

## Allowed Question Recipes
1. “Say: …” (production)
2. “Complete: …” (fill blank)
3. “Choose: …” (minimal)

## Hint Style
- short, calm, specific, never shame/rush

## ConceptTag Format
- a1.<topic>.<subtopic> (e.g., a1.greetings.hello)
